Render raw html in HUGO markdown files

By default Hugo will not render embedded html in markdown files. To enable, add the following lines to config.toml:

[markup.goldmark.renderer]
  unsafe = true

Entering the raw html:

<h1>hello world!</h1>

Renders:

hello world!
