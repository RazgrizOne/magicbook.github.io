GalliumOS Screenshot settings

By Andrew Adkins

January 31, 2022

*GalliumOS *uses xfce4 which is a Linux desktop environment.

xfce4 uses something called xfce screenshooter which is the built in screenshot application. (it may actually be seperate but it comes with GalliumOS)

You can go to keyboard settings in GalliumOS and change the command that is run to screenshot a selection instead of the entire screen.

In my case I used the command xfce4-screenshooter -c -r
