Hello#

This site is automatically generated as I type out my notes. There are a couple of steps to reach this goal, but they are not many, and the lessons learned while doing this will help you dip your toe into a computer science tool or two.

Open Chromium#

Let’s start simple.

The first thing you are going to do is open chromium in the bottom left corner of the laptop. This step is so easy you may have already done so.

chromium is the icon on the right of the picture above.

Download Obsidian#

Next, navigate to https://obsidian.md/

download the app image. This will land in your downloads folder. ‘/home/username/downloads’

Give the file permission to run#

By default, app images do not have permission to run code. You must give this permission manually by right clicking (or in the case of this laptop, tapping with two fingers) on the app image, then going to the ‘permissions’ page and checking the box next to ‘allow this file to run as a program’.

Once you have done this you will be able to run the program by double clicking it. At this point it is recommended that you move this file to your desktop if you have not already.

Run the program#

When you first run Obsidian, you are going to be asked to create a Vault

I always create my ‘vaults’ in a folder named obsidian inside of the documents folder. I tend to think of the documents folder as the save data folder for different software I use on my computer.

Once you have a vault created here are some things to try out.

Guide to markdown#

Here are some things to try

Page Linking#

This first technique is not actually markdown. Instead it is something obsidian does, but I find it very useful and unlike any other implementation of markdown or note taking software in general.

[[some page]]

some page

As you can see above, typing out ‘some page’ contained inside of double brackets, created a new page with the name ‘some page’. With this you can easily link many ideas together. If you then go into ‘some page’ and change the name from that page, it will change everywhere it was referenced saving a lot of work. When you are in obsidian, you can then CTRL + Click to enter this page and edit it.

Code blocks#

Note that the ` symbol is right under your escape key

```<language> (optional) write code here ```

for example…

```javascript const num1 = 5; const num2 = 3;

// add two numbers const sum = num1 + num2;

// display the sum alert(‘The sum of ' + num1 + ' and ' + num2 + ' is: ' + sum); ```

const num1 = 5;
const num2 = 3;

// add two numbers
const sum = num1 + num2;

// display the sum
alert('The sum of ' + num1 + ' and ' + num2 + ' is: ' + sum);

Essentially, code highlighting will be directed by what language you declare. If you don’t enter a language, you can also use it as a type of styling of the text.

Embedding iFrames#

You can embed other websites. For example…

You could embed a youtube video

<iframe width="560" height="315" src="https://www.youtube.com/embed/pgW4LWeUyVY" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
